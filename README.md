# multivac-nlp
Testing and benchmarking some of the existing NLP libraries in Apache Spark

## NLP libraries used in Multivac-NLP
### spark-nlp
latest version: [https://github.com/JohnSnowLabs/spark-nlp]()
### Stanford-CoreNLP
